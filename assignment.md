# Azure Key Vault Security Lab Assignment

## Objective

Implement Azure Key Vault security using Microsoft recommended practices.

---

## Tasks Completed

✔ Created Azure Key Vault

✔ Created Secret

✔ Enabled Managed Identity

✔ Assigned RBAC Roles

✔ Retrieved Secret using PowerShell

✔ Configured Networking

✔ Enabled Diagnostic Settings

✔ Reviewed Microsoft Defender Recommendations

---

## Observations

Azure Key Vault provides secure storage for secrets, keys and certificates.

Managed Identity removes the need for storing credentials inside applications.

RBAC provides secure authorization.

Diagnostic Settings help monitor all Key Vault activities.

Microsoft Defender for Cloud provides security recommendations.

---

## Conclusion

Azure Key Vault significantly improves cloud security by protecting sensitive data and reducing credential exposure.
