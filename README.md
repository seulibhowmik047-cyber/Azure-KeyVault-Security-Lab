# Azure Key Vault Security Lab

## Overview

This repository contains my hands-on Azure Key Vault Security Lab completed during Microsoft Azure Cloud Security training.

The lab demonstrates how Azure Key Vault secures application secrets, cryptographic keys, and certificates using Microsoft security best practices.

---

## Objectives

- Create Azure Key Vault
- Store Secrets securely
- Configure Managed Identity
- Configure Azure RBAC
- Retrieve Secrets securely
- Configure Networking
- Enable Diagnostic Settings
- Explore Microsoft Defender for Cloud

---

## Technologies

- Microsoft Azure
- Azure Key Vault
- Azure RBAC
- Managed Identity
- Azure Monitor
- Log Analytics Workspace
- Microsoft Defender for Cloud
- PowerShell

## Lab Architecture

Application
↓

Managed Identity

Azure Key Vault

Secrets / Keys / Certificates

---

## Best Practices

- Never store secrets in source code
- Use Managed Identity
- Use Azure RBAC
- Rotate Secrets regularly
- Enable Logging
- Enable Microsoft Defender for Cloud
- Use Private Endpoint whenever possible
